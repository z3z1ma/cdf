# P3 B8: secure streaming XML record codec

## Scope

Add namespace-aware streaming XML record extraction with pinned absolute record path, attribute/text/child mapping, bounded discovery, and mandatory DTD/entity/XInclude/network disablement.

## Acceptance criteria

- No whole-document DOM or external resolution occurs.
- Namespace, repeated/mixed content, attributes, depth/text/record limits, malformed input, and row quarantine match catalog semantics.
- Preview/run and local/remote/compressed paths are identical.
- Security corpus, memory ceiling, and native reference ratio are green.

## Evidence expectations

Dependency/security review, XXE/billion-laughs corpus, namespace/mixed-content goldens, fuzzing, memory, and profiles.

## Explicit exclusions

No arbitrary XPath/XSLT.

## Blockers

Depends on transforms, FX1, and L5.
