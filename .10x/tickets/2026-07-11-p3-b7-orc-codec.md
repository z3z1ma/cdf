# P3 B7: ranged stripe-parallel ORC codec

## Scope

Add native ORC footer/stripe discovery, deterministic stripe units, projection/predicate pushdown, compression, physical/logical type provenance, and local/remote ranged decode.

## Acceptance criteria

- Stripe concurrency/order and whole-file state semantics match the catalog.
- Pushdown fidelity is declared and rechecked when inexact.
- All supported ORC types map losslessly or fail through reconciliation with exact diagnostics.
- Same-host reference ratio, memory, range overlap, jobs invariance, and malformed behavior are green.

## Evidence expectations

Dependency review, ORC reference corpus/tool comparison, pushdown/type matrix, fuzz/malformed stripes, remote ranges, and profiles.

## Explicit exclusions

No Hive metastore/table semantics.

## Blockers

The named CDF dependencies are done. The only credible native Arrow implementation,
`orc-rust` 0.8.0, does not yet provide the resource-authority surface required by the
active codec-runtime spec: its async reader retains complete projected stripe streams,
its decoder allocates from untrusted length streams, and several compressors can exceed
the postscript block-size declaration before CDF can reconcile memory. Adding a wrapper
would account source ranges but would not contain dependency-owned decompression or
nested output. Closure requires bounded upstream decode APIs or a measured generic
process-isolated codec execution mode; a B7-local shim is prohibited.

## Journal

- 2026-07-18: Evaluated `orc-rust` 0.8.0 and the DataFusion 54 `datafusion-orc`
  adapter before adding dependencies. The crate has the correct semantic shape—tail
  metadata, public stripe metadata, async exact ranges, projection, byte-range stripe
  selection, row-index predicates, and all ORC compression/type families. The
  DataFusion adapter is only an object-store wrapper over the same reader, not a
  different engine. Source inspection found the resource-authority defects recorded
  in `.10x/research/2026-07-18-orc-native-decoder-admission.md`; B7 is blocked rather
  than introducing another known-unbounded production decoder.
