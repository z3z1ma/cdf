# Generated error catalog

The error catalog is intentionally not hand-written in WS6A.

WS4 owns the stable error-code catalog, and WS6B owns generating these docs from
that source of truth:
[`2026-07-08-p1-product-ws6b-generated-reference-freshness.md`](../../.10x/tickets/2026-07-08-p1-product-ws6b-generated-reference-freshness.md).

Until then, operator guides may describe current failure modes in prose, but
stable error codes, taxonomy entries, remediation fields, and exit-code mappings
must come from the generated catalog.
